块级作用域

我们可以将let看成更完美的var
变量作用域：变量在什么范围内是可用的

let

  1. let 所声明的变量，只在let命令所在的代码块内有效
  2. let 不允许在相同作用域内，重复声明同一个变量。

    const

  3. 注意一：const声明一个只读的常量。一旦声明，常量的值就不能改变。
  4. 注意二：在使用const定义标识符，const声明的变量不得改变值，这意味着，const一旦声明变量，就必须立即初始化，不能留到以后赋值。
  5. 注意三：常量的含义是指向的对象不能修改，但可以改变对象的内部属性,const声明的常量，也与let一样不可重复声明。

Promise

  • Promsie到底是做什么的呢?
    • Promise是异步编程的一种解决方案
    • 那什么时候我们会处理异步事件呢?
      • 一种常见的场景应该就是网络请求了
      • 但是当网络请求非常复杂时,就会出现回调地狱
      • 使用Promise对这个异步操作进行封装
    • 语法:
      • new -> 构造函数(1.保存了一些状态信息 2.执行传入的函数)
      • 在执行传入的回调函数时,会传入两个参数,resolve,reject,它们两个本身又是函数
        Promise第一种写法
  • Promise的三种状态
    • 首先,当我们开发中有异步操作时,就可以给异步操作包装一个Promise
    • 异步操作之后有三种状态
      • pending:等待状态,比如正在进行网络请求,或者定时器没有到时间
      • fulfil:满足状态,当我们主动回调了resolve时,就处于该状态,并且回调.then()
      • reject:拒绝状态,当我们主动回调了reject时,就处于该状态,并且会回调.catch()
        Promise第二种写法
  • Promise链式调用
  • promise.resolve():将数据包装成Promise对象,并且在内部回调resolve()函数
  • promise.reject():将数据包装成Promise对象,并且在内部回调reject()函数
    简写 return new Promise((resolve,reject) => {}) -> return Promise.resolve(res) -> return res
    //return 等价于 resolve throw 等价于 reject
