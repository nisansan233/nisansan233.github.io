一. 组件化开发

组件的使用分为三大步

  • 创建组件构造器 调用Vue.extend()方法
  • 注册组件 调用Vue.component()方法
  • 使用组件

全局组件和局部组件

  • 当我们通过调用Vue,component()注册组件时，组件注册是全局的
    这意味着该组件可以在任意Vue示例下使用

  • 在vue实体里定义component为局部组件，只能在el挂载的元素中使用

  • 父子组件错误用法：以子标签形式在Vue实例中直接使用

    • 因为当子组件注册到父组件的components时，Vue会编译好父组件的模块

    • 类似这种用法，<child-cpn><child-cpn>是会被浏览器忽略的

组件中的数据存放

组件是一个单独功能模块的封装，不能直接调用Vue实例中的data属性，子组件不能直接访问父组件

这个模块有属于自己的HTML模板，也应该有属性自己的data
组件对象也有一个data属性（也可以有methods等属性）
data属性必须是一个函数
而且这个函数返回一个对象，对象内部保存着数据,组件data只在本地范围内有效，采用函数式返回的形式，这样是为了防止跨组件变量污染

组件通信

父传子 props()

  • 在组件中，使用选项props来声明需要从父级接收到的数据
  • props的值有两种方式：
    1.方式一：字符串数组，数组中的字符串就是传递时的名称
    2.方式二：对象，对象可以设置传递时的类型，也可以设置默认值等 props数据验证
    支出类型：
    String
    Number
    Boolean
    Array
    Object
    Date
    Function
    Symbol

子传父（自定义事件）
自定义事件流程：

  • 在子组件中，通过$emit()来触发事件
  • 在父组件中，通过v-on来监听子组件事件
    $emit(事件，参数)传递需要监听的事件和参数，v-on为事件绑定父组件中的方法,注意绑定时不要给方法加括号

案例：子组件定义计数器按钮，父组件进行数据展示

使用watch属性对事件进行监听

1.1. 父子组件的访问

  • 有时候我们需要父组件直接访问子组件，子组件直接访问父组件，或者是子组件访问跟组件。

    • 父组件访问子组件：使用$children或$refs
    • 子组件访问父组件：使用$parent

  • 我们先来看下$children/$refs的访问

  • this.$children是一个数组类型，它包含所有子组件对象。
    我们这里通过一个遍历，取出所有子组件的name状态。

  • $children用的很少，需要拿到所有子组件时才使用

  • 一般使用$refs,
    对象类型，默认是一个空的对象
    使用时需要在组件标签上添加ref属性，比如ref=’zzz’

子组件访问父组件： parent/root

  • 使用$parent,访问根组件$root

1.2. slot的使用

组件的插槽

组件的插槽也是为了让我们封装的组件更加具有扩展性。
让使用者可以决定组件内部的一些内容到底展示什么。
抽取共性，保留不同
  • 基本使用

1.插槽的基本使用<slot></slot>
2.插槽可以设置默认值<slot>button</slot>
例如在solt里设置默认值是一个button标签
3.如果有多个值，同事放入组件中进行替换，会一起作为替代元素

  • 具名插槽
    给solt元素设置一个name属性，用来封装独立组件

  • 编译的作用域

  • 父组件模板的所有东西都会在父级作用域内编译；子组件模板的所有东西都会在子级作用域内编译

  • 作用域插槽

  • slot-scope=”自定义属性名”

  • vue3.0
  • v-slot:插槽名=”自定义属性名”
    v-slot 只能添加到 <template> 上,不能像slot那样添加到任意元素标签中
    注意message是所有你绑定属性的集合，也就是你写的：aa=“title”会当做message的属性来实现。当然这里message可以换做其它的名称。
