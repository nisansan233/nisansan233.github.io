五.div和span元素

  • span元素
    • 默认情况下,和普通文本几乎没有区别
    • 用于区分特殊文本和普通文本，比如用来显示一些关键字
    • 文本信息在一行展示，行内标签 内联标签
  • div元素
    • 一般作为其他元素的父容器，把元素包住，代表一个整体
    • 用于把网页分割成多个独立的部分
    • 每一个div占满一整行。块级标签

  • <header>:页眉

十. HTML-列表元素:

  • HTML提供了3组常用的用来展示列表的元素
    • 有序列表:ol/li
      • ol直接的直接元素只能是li
    • 无序列表:ul/li
      • ul直接的直接元素只能是li
    • 定义列表:dl/dt/dd
      • dl定义列表,直接子元素只能是dt、dd
      • dt列表中的每一项的项目名
      • dd列表中每一项的具体描述,是对dt的描述、解释、补充
  • 列表元素CSS属性
  • 它们都可以继承,所以设置给ol、ul元素,默认也会应用到li元素
    • list-style-type:设置li元素前面标记的样式
      • disc(实心圆)、circle(空心圆)、square(实心方块)
      • decimal(阿拉伯数字)、lower-roman(小写罗马数字)、upper-roman(大写罗马数字)
      • none(什么也没有)
    • list-style-image:设置某张图片为li元素前面的标记,会覆盖list-style-type的设置
    • list-style-position:设置li元素前面标记的位置,可以取outside,inside两个值
    • list-style:是list-style-type,list-style-image,list-style-position缩写属性
      • 最常用的设置是none

十一.HTML-表格元素:

  • table常见的元素

    • table 表格
      • border 边框的宽度
      • cellpadding 单元格内部的边距
      • cellspacing 单元格之间的间距
      • width 表格的宽度
      • align 表格的水平对齐方式
        • left
        • center
        • right
    • tr 表格中的行
      • valign 单元格的垂直对齐方式
        • top
        • middle
        • bottom
        • baseline
      • align 单元格的水平对齐方式
        • left
        • center
        • right
    • td 行中的单元格
      • valign
      • align
      • width
      • height
      • rowspan:单元格可横跨的行数
      • colspan:单元格可横跨的列数

  • 单元格边框的合并(细线表格)

    • border-collapse:collapse

  • 其他元素

    • caption:表格的标题
    • thead:表格的表头
    • tbody:表格的主体
    • tfoot:表格的页脚
    • th:表格的表头单元格

  • 单元格的合并

    • colspan:单元格可横跨的列数
    • rowspan:单元格可横跨的行数

  • border-spacing

    • CSS属性,用于设置单元格之间的水平、垂直间距比如
      table{border-spacing: 10px 20px }

十二.HTML-表单元素

  • form 表单

    • 一般情况下,其他表单相关元素都是它的后代元素

  • input 单行文本输入框、单选框、复选框、按钮等元素

    • type:input的类型
      • text
      • password
      • button
      • radio
      • checkbox
      • file
      • reset
      • submit

  • select、option 下拉选择框

    • select常用属性
      • multiple:可以多选
      • size ：显示多少项
    • option常用属性
      • selected:默认被选中

  • button 按钮实现方式

    • input
    • button

  • label 表单元素的标题

    • label元素中的for属性绑定input元素中的id属性

  • 去除input的Tab键选中效果

    • 在CSS属性outline设置为none
    • tabindex=-1

  • textarea 多行文本框

    • cols:列数
    • rows:行数
    • 缩放的CSS属性设置
      • 禁止缩放:resize:none
      • 水平缩放:resize:horizontal
      • 垂直缩放:resize:vertical
      • 水平垂直缩放:resize:both(默认值)

  • fieldset 表单元素组

  • legend fieldset的标题
    fieldset展示

    input其他属性

  • maxlength:容许输入的最大值

  • placeholder:占位文字

  • readonly:只读

  • disabled:禁用

  • checked:默认被选中

    • 只有当type为radio或者checkbox时可用

  • autofocus:当页面加载时,自动聚焦

  • name:名字

    • 在提交数据给服务器时,可用于区分数据类型

  • value:取值

  • form:设置所属的form元素(填写form元素id)

  • 一旦使用了此属性,input元素即使不在form内部,它的数据也能够提交给服务器

  • form常用属性

    • action
      • 用于提交表单数据的URL
    • method
      • 请求方法(get和post),默认是get
        • get请求参数会在地址栏中显示。会封装到请求行中(HTTP协议后讲解)。请求参数大小是有限制的,不太安全。
        • post请求参数不会再地址栏中显示。会封装在请求体中,请求参数的大小没有限制。 较为安全。
    • target
      • 在什么地方打开URL(参考a元素的target)
    • enctype
      • 规定了在向服务器发送表单数据之前如何对数据进行编码
      • 取值有三种
        • appliction/x-www-form-urlencoded:默认编码方式
        • multipart/form-data:文件上传时必须为这个值,并且method必须是post
        • text/plain:普通文本传输
    • accept-charset
      • 规定表单提交时使用的字符编码(默认值UNKNOWN,和文档相同编码)

十三.元素类型

  • 显示类型

    • 块级元素
      • 独占父元素一行
      • 比如div,p,pre,h1~h6,ul,ol,li,dl,dt,dd,table,form,article,aside,footer,header,hgroup,main,nav,section,blockquote,hr等
    • 行内级元素
      • 多个行内级元素可以在父元素的同一行中显示
      • 比如a,img,span,strong,code,iframe,label,input,button,canvas,embed,object,video,audio等

  • 内容类型

    • 替换元素-行内级元素
      • 元素本身没有实际内容,浏览器根据元素的类型和属性,来决定元素的具体显示内容
      • 比如img,input,iframe,video,embed,canvs,audio,object等
    • 非替换元素
      • 和替换元素相反,元素本身是有实际内容的，浏览器会直接将其内容显示出来，而不需要根据元素类型和属性类型来判断到底显示什么内容

  • 行内级非替换元素注意事项

    • 以下属性对行内级非替换元素不起作用
      • width,height,margin-top,margin-bottom
    • 以下属性对行内级非替换元素效果特殊
      • padding-top,padding-bottom,上下方向border
      • 增加的上下内边距和边框会多出来区域，但是不占据空间
